# Frequently Asked Questions (FAQ) Website

A modern and responsive FAQ website developed using HTML, CSS, and JavaScript.

## Features
- Expand and collapse FAQ answers
- Smooth animations and transitions
- Clean and responsive design
- User-friendly interface
- Mobile-friendly layout

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose
This project helps users quickly find answers to common questions through an organized and interactive FAQ section.
